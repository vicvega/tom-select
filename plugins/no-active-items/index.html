No Active Items

Prevents users from selecting items (eg selected options). Normally, users can active items by either clicking on them, shift+clicking, shift+left-arrow. This plugin disables that functionality.

  • JavaScript
  • HTML
  • Edit
    • JSFiddle
    • CodePen 
new TomSelect('#ex-no-active-items',{
	plugins: ['no_active_items'],
	persist: false,
	create: true,
});
<input type="text" id="ex-no-active-items" value="awesome,neat" autocomplete="off">

No additional configuration settings for this plugin
